Editor Manual Documentation

NOTE: This website using Markdown as Formating Editor so it won't support ASCII Character and some styling. Here is documentation about general ASCII Character and style that can be applied:


ASCI Code (Add & in front of the code)

  • < : lt;
  • > : gt;
  • & : amp;
  • ¥ : yen;

Styling

  • centering text: <p style="text-align:center"></p>
  • horizontal rule: <hr />
